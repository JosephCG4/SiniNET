Sistema web para la gestión de siniestros de una aseguradora, que permite registrar, consultar y administrar información relacionada con siniestros, clientes y procesos internos.
## Funcionalidades
- Registro y gestión de siniestros
- Administración de clientes
- Gestión de información asociada a pólizas
- Interfaz web para usuarios administrativos
- Arquitectura separada por capas (Backend, Frontend y WS)

## Arquitectura
El proyecto está organizado en capas claramente separadas:

- Backend: lógica de negocio y acceso a datos
- Frontend: interfaz web para el usuario

Esta separación permite un código más mantenible, escalable y fácil de entender.

## Tecnologías utilizadas
- Lenguaje: C# / Java / JavaScript
- Backend: ASP.NET / Spring Boot
- Frontend: HTML, CSS, JavaScript
- Librerías: Bootstrap, jQuery
